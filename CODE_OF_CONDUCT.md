# 贡献者行为准则

## 我们的承诺

作为社区成员、贡献者和维护者，我们承诺让每个人都能参与到本项目和社区中，且不受骚扰，无论其年龄、体型、可见或不可见的残障、族裔、性别特征、性别认同与表达、经验水平、教育程度、社会经济地位、国籍、外貌、种族、宗教或性取向如何。

我们承诺以有助于建设开放、友善、多元、包容和健康社区的方式行事和互动。

## 我们的准则

有助于营造积极环境的行为包括：

- 对他人展现同理心和善意。
- 尊重不同的观点和经验。
- 给予并优雅地接受建设性反馈。
- 为自己的错误承担责任，向受影响者致歉，并从中学习。
- 关注对整个社区而非仅对个人最有利的事。

不可接受的行为包括：

- 使用带有性暗示的言语或图像，以及任何形式的性关注或挑逗。
- 挑衅、侮辱或贬损性评论，以及人身或政治攻击。
- 公开或私下的骚扰。
- 未经明确许可，公布他人的私人信息（如实际地址或电子邮箱）。
- 在专业场合中可合理认定为不当的其他行为。

## 执行责任

社区维护者有责任阐明并执行可接受行为的准则，并对任何他们认为不当、具威胁性、冒犯性或有害的行为采取适当且公正的纠正措施。

## 适用范围

本行为准则适用于所有社区场合，也适用于个人在公开场合代表社区时的情形。

## 执行

如遇滥用、骚扰或其他不可接受的行为，可通过 GitHub 的私密报告通道（本仓库 **Security** 标签页的 **Report a vulnerability**）联系负责执行的维护者。所有投诉都会被及时且公正地审查和调查。维护者有义务为事件报告者保密。
