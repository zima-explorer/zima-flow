# zimaflow

This is the public entrypoint for zimaflow. Use it when an agent can read this repository directly or when a user explicitly points to this file.

Do not infer the workflow from this entrypoint alone. Choose the matching source skill below, read it completely, then follow its instructions and any referenced files.

## Notes

- `skills/` is the source form. Keep these files readable by agents that support explicit paths or recursive directory access.
- Runtime adapters such as `zimaflow-<name>/SKILL.md` are generated by `scripts/install.sh --adapter-dir` only for hosts that require one-level skill discovery.
- Keep `rules/` and `references/` next to `skills/`, and set `ZIMAFLOW_HOME` to this repository or the installed target directory when using copied sources.
